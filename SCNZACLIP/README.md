## 🧠 HOW IT WORKS (หลักการทำงาน) 🛠️

ระบบทำงานตามวงจร 6 ขั้นตอนนี้วนไปเรื่อยๆ:

1.  **🕵️ SCAN:** บุกเข้าไปสำรวจเป้าหมาย -> 🐙 **Scanzaclip**
2.  **🧱 DETECT:** ตรวจพบสภาพแวดล้อม (เจอกำแพง? ทางเปิด? ข้อมูล?) -> 🧠 **VIDER**
3.  **⚖️ DECIDE:** จัดลำดับความสำคัญ ตัดสินใจว่าต้องทำอะไรก่อน -> 🧠 **VIDER**
4.  **🗝️ SOLVE:** **ถ้าเจอกำแพง = สร้างกุญแจ!** -> ✨ **VIDER KeyGen**
5.  **🚪 PENETRATE:** ใช้กุญแจเปิดทาง บุกเข้าไปข้างใน -> 🚀 **Penetration Engine**
6.  **🔁 REPEAT:** วนกลับไปข้อ 1 **จนกว่าจะครบ 100%** ✅
